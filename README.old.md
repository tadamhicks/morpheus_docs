# morpheus-docs

## download
```
$ git clone git@github.com:gomorpheus/morpheus-docs.git
```

## build
```
$ cd morpheus-docs
$ ./gradlew asciidoctor
```

## view
asciidoctor will compile .adoc files to .html files in `build` directory.  Open up `index.html` in your browser of choice to view
